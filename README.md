# Mohit Narula - Personal Website

A modern, professional dark mode personal website with clean navigation and easy-to-follow links.

## Features

- **Dark Mode Design**: Professional dark theme with carefully chosen colors
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Easy-to-follow links with active state indicators
- **Sections**: Home, About, Projects, Shop, and Contact
- **Social Media Integration**: Links to YouTube, Spotify, LinkedIn, Twitter, and Facebook
- **Modern Animations**: Smooth scroll, fade-in effects, and hover interactions

## File Structure

```
mohitnarula_website/
├── index.html      # Main HTML structure
├── styles.css      # Dark mode professional styling
├── script.js       # Navigation and interaction functionality
├── _DSC4393.jpg    # Profile image (you'll need to add this)
└── README.md       # This file
```

## Setup

1. **Add Your Profile Image**
   - Place your profile image (`_DSC4393.jpg`) in the root directory
   - Or update the image path in `index.html` if using a different filename

2. **Update Social Media Links**
   - Edit the social media links in `index.html` with your actual URLs
   - Update the `href` attributes for each social platform

3. **Customize Content**
   - Update the About section with your personal information
   - Add your projects to the Projects section
   - Customize the Shop section as needed

4. **Deploy**
   - Upload all files to your web hosting
   - Or deploy to GitHub Pages, Netlify, Vercel, etc.

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --accent-primary: #6366f1;  /* Change primary accent color */
    --bg-primary: #0a0a0a;      /* Change background color */
    /* ... more variables */
}
```

### Sections
Add or remove sections by:
1. Adding HTML in `index.html`
2. Adding corresponding styles in `styles.css`
3. Updating navigation links
